# Game Dadu Babi
Game Dadu Babi adalah permainan dadu dimana salah satu pemain harus mencapai angka 100 untuk memenangi game. Game Dadu Babi ini menggunakan 6 buah dadu dan sangat cocok dimainkan bersama para member.
